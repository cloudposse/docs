Goofys is a utility that implements S3-backed filesystems using FUSE.

[`goofys`](/learn/toolchain/#geodesic) is a a high-performance, POSIX-ish Amazon S3 file system written in Go. We use `goofys` in [Geodesic](/reference/tools.mdx#geodesic)
