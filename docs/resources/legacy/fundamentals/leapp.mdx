:::warning Deprecated
Leapp is deprecated. We now use **Atmos Auth** for AWS authentication.

See [How to Log into AWS](/layers/identity/how-to-log-into-aws/) for the current approach.
:::

[https://github.com/Noovolari/leapp](https://github.com/Noovolari/leapp)

Leapp was a Desktop Dev Tool that handled the management and security of cloud credentials. It has been replaced by **Atmos Auth**, which provides seamless authentication via IAM Identity Center.
